# Exercism

Client gem for the warmup-exercise app exercism.io.

    $ gem install exercism

## Usage

    $ exercism login YOUR_GITHUB_USERNAME -s YOUR_EXERCISM_SECRET
    $ exercism fetch
    $ exercism submit -f example.rb
